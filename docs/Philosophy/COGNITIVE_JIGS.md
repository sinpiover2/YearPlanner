# Cognitive Jigs

## Why this document exists

Year Planner is not fundamentally a lesson planning application.

It is an attempt to build cognitive tools that reduce unnecessary mental effort so teachers can spend more attention on teaching.

This document describes that philosophy.

It should change very rarely.

---

# The Insight

A woodworking jig does not build furniture.

It removes unnecessary effort while preserving the craft.

The craftsperson still decides.

The jig simply makes good work easier.

Year Planner exists for the same reason.

It should never replace teaching.

It should remove repetitive mental work so the teacher can think more deeply about students and instruction.

---

# Humans are extraordinary and limited

Humans have remarkable strengths.

- Judgment
- Creativity
- Empathy
- Improvisation
- Pattern recognition
- Meaning making

Humans also have well-known limitations.

- Limited working memory
- Easily distracted
- Forgetful
- Fatigue
- Decision overload
- Context switching

Good tools acknowledge both.

The purpose of Year Planner is not to eliminate human judgment.

Its purpose is to support it.

---

# A Cognitive Jig

A Cognitive Jig reduces thinking that should not require thought.

Examples:

Instead of remembering what was taught last Tuesday...

? show it.

Instead of remembering which sections still need copies...

? remind gently.

Instead of calculating remaining instructional time...

? visualize it.

Instead of reconstructing a lesson from memory...

? preserve teaching episodes.

The teacher still decides.

The jig simply removes unnecessary cognitive load.

---

# What the software should optimize

Not clicks.

Not forms.

Not documentation.

Not reports.

Instead:

Reduce unnecessary cognitive effort.

Every feature should answer one question.

"What mental work does this remove?"

If the answer is "none,"

the feature probably does not belong.

---

# Arrange vs Compose

Teaching contains different kinds of thinking.

Arranging.

Composing.

Teaching.

Reflecting.

Each deserves its own workspace.

A cognitive jig supports one mode of thinking at a time.

It should not ask the teacher to arrange while composing or reflect while teaching.

---

# Curriculum is reference

Curriculum is not authored work.

Curriculum informs thinking.

Teaching Episodes represent thinking.

The software should always give visual priority to the teacher's authored work.

Reference material should remain available without competing for attention.

---

# Reflection creates teaching wisdom

Lesson plans are temporary.

Teaching wisdom accumulates.

Reflection is where experience becomes knowledge.

Year Planner should preserve teaching wisdom rather than simply archive lesson plans.

---

# Design Test

Whenever considering a new feature, ask:

Does this remove unnecessary cognitive effort?

Does this preserve teacher judgment?

Does this make thinking easier rather than documenting more?

Would this feel like a good woodworking jig?

If not,

keep designing.

---

# One sentence

Year Planner is a collection of cognitive jigs that help teachers think, remember, and organize so they can devote more attention to teaching students.
