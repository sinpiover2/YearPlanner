Phase 0 Đ Philosophy

? Complete

Established the foundational principles of Year Planner.

Key ideas:

* Year Planner is a teacher decision-support tool.
* The organizing question is:
    Am I OK?
* Default emotional state is calm.
* Information order is:
    1. Reality
    2. Consequence
    3. Recommendation
* Timeline = orientation.
* Cards = interpretation.
